# ImgVault Knowledge Base

This folder captures project knowledge for day-to-day development and debugging.

## Documents

- `project-overview.md`: high-level map of the monorepo and responsibilities
- `architecture.md`: how extension, native host, and web app fit together
- `extension-workflows.md`: extension save/upload/download flows and key modules
- `native-host-yt-dlp.md`: native host behavior, yt-dlp strategy, and fallback logic
- `web-app-api.md`: Next.js app and API behavior, including Firestore mapping
- `known-gotchas.md`: operational pitfalls and what to check first
- `build-and-release.md`: practical build commands and release checklist

## Scope

These notes are based on the current code under:

- `nextgen-extension`
- `native-host`
- `web`

The `old extension` folder is legacy/reference only.
