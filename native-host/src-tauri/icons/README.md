# Icon Placeholder

This directory should contain the following icon files for the Tauri app:

- `32x32.png` - 32x32 pixel PNG icon
- `128x128.png` - 128x128 pixel PNG icon
- `128x128@2x.png` - 256x256 pixel PNG icon (for retina displays)
- `icon.icns` - macOS icon file
- `icon.ico` - Windows icon file

You can use any image editor to create these icons from your ImgVault logo.

For now, the app will build without icons (Tauri will use defaults).
